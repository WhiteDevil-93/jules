# Jules Extension for VSCode

[![VSCode Extension](https://img.shields.io/badge/VSCode-Extension-blue.svg)](https://marketplace.visualstudio.com/items?itemName=HirokiMukai.jules-extension)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

> "Experience the future of coding with Google Jules in VSCode"

Jules Extension is an extension that allows you to operate Google's AI coding agent **Jules** directly from within VSCode.
Welcome an intelligent partner to your coding workflow.

## ‚ú® Concept

This extension was created to take your development experience to the next level.

- **Seamless Integration:** Access the powerful features of Jules without leaving your usual VSCode environment.
- **Real-time Collaboration:** From creating a coding session to checking its progress, everything is in real-time.
- **Productivity Leap:** Leave the tedious tasks to Jules and focus on your creative work.

## üì¶ Installation

Install from the [Visual Studio Code Marketplace](https://marketplace.visualstudio.com/items?itemName=HirokiMukai.jules-extension)

Or search for "Jules Extension" in VS Code Extensions view.

### From Marketplace (Recommended)

1. Search for "Jules Extension" in the VSCode Marketplace
2. Click the `Install` button

### From VSIX File (Manual Install)

If you want to try the latest features that are not yet published on the Marketplace, you can download and install the `.vsix` file directly from the releases page.

1. **Go to the Releases Page:**
   Visit [GitHub Releases](https://github.com/is0692vs/jules-extension/releases) and find the latest release version.

2. **Download the VSIX File:**
   Download the `.vsix` file (e.g., `jules-extension-0.1.0.vsix`) from the `Assets`.

3. **Install in VSCode:**
   - Open VSCode.
   - Go to the `Extensions` view (`Ctrl+Shift+X`).
   - Click the `...` (More Actions) menu at the top of the view and select `Install from VSIX...`.
   - Select the downloaded `.vsix` file to install.

## üîë Getting Your API Key

To use the Jules Extension, you need a Jules API key. Follow these steps to get one:

1. **Create an Account:**

   - Go to the [Jules Official Website](https://jules.google/docs).
   - Sign up for a new account or log in if you already have one.

2. **Generate API Key:**

   - Navigate to the "API Keys" or "Developer Settings" section in your account dashboard.
   - Click on "Create a new secret key".
   - Give your key a descriptive name (e.g., "VSCode Extension") and generate it.

3. **Copy Your Key:**
   - Your new API key will be displayed. Copy it to your clipboard.
   - If you need to view your key again later, you can always find it on your Jules settings page.

> **Important:** Treat your API key like a password. Do not share it publicly or commit it to version control.

## Authentication

For Jules to interact with your GitHub repositories, you need to authenticate your GitHub account.

### Method 1: OAuth Sign-in (Recommended) ‚úÖ

This is the easiest and most secure method.

1.  Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`).
2.  Run the **`Jules: Sign in to GitHub`** command.
3.  You will be redirected to your browser to authorize the extension.

### Method 2: Personal Access Token (PAT) (Deprecated) ‚ö†Ô∏è

**Using PATs is deprecated and this functionality will be removed in a future version.** Please migrate to the OAuth sign-in method.

## ‚öôÔ∏è Extension Settings

This extension contributes the following settings, which can be configured in the VSCode settings UI (`Ctrl + ,` or `Cmd + ,`):

- **`jules-extension.apiKey`**: Your Jules API key. It's recommended to set this via the `Jules: Set Jules API Key` command, which stores it securely.
- **`jules-extension.autoRefresh.enabled`**: Automatically refresh the sessions list.
  - *Default:* `false`
- **`jules-extension.autoRefresh.interval`**: The interval in seconds for auto-refresh.
  - *Default:* `30`
  - *Minimum:* `10`
- **`jules-extension.customPrompt`**: A persistent, custom instruction to prepend to every message sent to Jules. Useful for setting context or specifying a response format.
  - *Example:* `Always reply in Japanese.`
- **`jules-extension.hideClosedPRSessions`**: Automatically hide sessions associated with closed or merged pull requests from the session list.
  - *Default:* `true`
- **`jules.defaultBranch`**: The default branch selection behavior when creating a Jules session.
  - *`current`*: (Default) Use the current Git branch.
  - *`default`*: Use the repository's default branch.
  - *`main`*: Use `main`, or the repository's default if `main` does not exist.

### Deprecated Settings

- **`jules-extension.githubToken`** and **`jules.githubPat`**: These settings are deprecated. Please use the `Jules: Sign in to GitHub` command for secure authentication instead.

## Quick Start

1.  Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`).
2.  Run **`> Jules: Sign in to GitHub`** and follow the browser prompts to authenticate.
3.  Run **`> Jules: Set Jules API Key`** and enter your API key.
4.  Click the `$(robot)` icon in the Activity Bar to open the Jules Sessions view.
5.  Click the `+` icon or run **`> Jules: Create Jules Session`** to start your first coding session!

## Commands

This extension provides the following commands in the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`):

- **`Jules: Sign in to GitHub`**: Securely authenticate with your GitHub account (Recommended).
- **`Jules: Set Jules API Key`**: Configure your API credentials.
- **`Jules: Verify Jules API Key`**: Test your API connection.
- **`Jules: Create Jules Session`**: Start a new coding session.
- **`Jules: List Jules Sources`**: Browse available data sources.
- **`Jules: Refresh Jules Sessions`**: Reload the sessions list in the sidebar.
- **`Jules: Show Jules Activities`**: View the detailed activity log for a selected session.
- **`Jules: Refresh Jules Activities`**: Update the activity view with the latest progress.
- **`Jules: Send Message to Jules Session`**: Send follow-up instructions to an active session.
- **`Jules: Approve Jules Plan`**: Review and approve a generated plan for execution.
- **`Jules: Open Jules Settings`**: Open the VSCode settings page for this extension.
- **`Jules: Delete Session from Local Cache`**: Remove a session from the sidebar view.
- **`Jules: Clear Jules Cache`**: Clear all locally cached extension data.
- **`[DEPRECATED] Jules: Set GitHub PAT`**: Set a GitHub Personal Access Token (deprecated).
